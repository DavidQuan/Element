## Description:

Element is a programatic UI framework for OSX. You can edit the UI design while your app is running. Element is built from the ground up and provides highly customizable UI components that you can remix to make a truly unique UI/UX experience. More information in the [Wiki](https://github.com/eonist/Element/wiki)
